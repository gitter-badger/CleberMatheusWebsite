Progresso do Revox Alpha 2

Até esta publicação já terminei de adicionar os protocolos: HTTPS, file, mailto, NNTP, wais e prospero. O Gumbo foi removido e o analisador html está em fase de desenvolvimento.

Atualmente estou desenvolvendo o suporte ao AppCache e o suporte ao elemento HTML, sendo que o suporte ao doctype está concluído. O Analisador HTML funcionará da seguinte maneira, devera analisar se uma tag está aberta, se não irá criar um elemento em uma lista para onde armazenará o conteúdo do elemento aberto até encontrar a tag final do elemento.

O Analisador HTML está sendo baseado do seguinte site: HTML5.1 Nightly - W3C

Código-Fonte do Revox: BCMUniverse\Revox - GitHub
