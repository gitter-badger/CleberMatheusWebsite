Introdução ao HTML

Quando você olha para uma página de Internet em um navegador, você enxerga, em um nível mais básico, palavras. Estas palavras geralmente tem algumas características de estilo, como diferentes tamanhos de fontes e cores. Em muitos casos uma página também exibe imagens ou talvez vídeo. Algumas vezes existe um formulário onde você pode inserir (ou procurar por) informação, ou personalizar a exibição da página ao seu gosto. Frequentemente uma página contém conteúdo animado e conteúdo que muda enquanto uma página permanece a mesma.

Várias tecnologias (como CSS, JavaScript, Flash, JSON) podem ser usadas para definir os elementos de uma página. Contudo, no nível mais baixo, uma página de Internet é definida usando HTML (HyperText Markup Language, ou Linguagem de Marcação de HiperTexto). Sem HTML não haveria página de Internet.

Este artigo fornece uma introdução ao HTML. Se você já se perguntou o quê acontece por trás das cenas em seu navegador de Internet, este artigo é o lugar para começar a aprender.
