Sobre SourceForge.net:

SourceForge.net é o maior fornecedor mundial de hospedagem para projetos de desenvolvimento de software de código aberto. SourceForge.net fornece uma variedade de serviços para projetos, incluindo uma rede de espelho de download, o desenvolvimento colaborativo de ferramentas (como CVS e Subversion) e ferramentas para apoiar a discussão e suporte. Estes serviços são fornecidos para projetos e seu usuários finais de gratuita.
