O que é OpenSource?

OpenSource (ou Código Aberto) é uma ramificação do movimento do Software Livre. Eles compartilham conceitos muito semelhantes, mas o OpenSource tenta fazer este método de desenvolvimento de programas compatível com o mundo empresarial. O foco é colocado no poder da comunidade para desenvolver software, e não em garantir a liberdade dos usuários. O OpenSource é definido por 10 pontos:

  • Livre redistribuição do software (comercial ou não).
  • Código-fonte aberto para ser estudado e modificado.
  • Obras derivadas são permitidas e devem ser distribuídas sob a mesma licença.
  • Integridade do código-fonte do autor, o que significa que trabalhos derivados devem ser diferenciados.
  • Sem discriminação contra pessoas ou grupos.
  • Sem discriminação contra os campos de atuação.
  • Distribuição de licença. Significa que todos beneficiam da licença do software.
  • A licença não deve ser específica a um produto. A licença é válida para programas de uma distribuição.
  • A licença não deve restringir outros softwares.
  • A licença deve ser tecnologicamente neutra.

OpenSource é hoje um modelo de desenvolvimento de software reconhecido e respeitado no ramo da tecnologia. Seus principais adversários são as patentes de software que proíbem os programadores a criar certos tipos de programas.

Para ter mais informações visite o site OpenSource.org.

Sobre OpenSource:

Um benefício para os usuários, software com código aberto é licenciado assim que você pode baixar e usar o software livre de encargos. O código fonte deste software é feito disponível livre de encargos, você (ou um programador que você contratar) pode fazer alterações a este software para melhor atender às suas necessidades, e você pode liberar seu código alterado de volta para a Comunidade, passando o benefício a outros usuários.
